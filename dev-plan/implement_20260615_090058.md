## 안전 불변식 (요약)

1. 기본값 전환은 **fail-closed 방향만** — 현재 통과 바를 낮추지 않는다.

5. PR 산출물은 **재해시·독립 재현·비용 상한**으로 봉인(Phase 3).
6. 자동 merge는 영구 금지.
